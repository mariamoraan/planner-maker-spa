## About the Project

This is a Single Page Application (SPA) for creating and customizing planner templates. Users can upload images, add various fields, and generate personalized planners.

## Features

- **Design your own planners** with fully customizable images, layouts, and templates.
- **Dynamic date management**: automatically update month covers, monthly calendars, and weekly schedules.
- **Export as PDF**: download your planner with optimized quality and minimal file size.
- **Multi-planner management**: create, edit, and organize multiple planners effortlessly.
- **Local data storage**: all your planners are saved directly on your device, so you can continue working later without losing progress - No login required !
- **Intuitive and user-friendly interface** for smooth planning and customization.

## Upcoming Features

- **Cloud sync**: save your planners online and access them from multiple devices.  
- **Collaboration**: share planners with others and collaborate in real-time.  
- **Template library**: browse and use pre-made planner templates.  
- **Advanced customization**: more layout options, stickers, and interactive elements.  
- **Mobile support**: optimized experience for tablets and phones.  

## Running Locally

To run the project locally:

1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`

The application will be available at `http://localhost:8080`.
